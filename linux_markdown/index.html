Markdown笔记

简介

  • Markdown是一种轻量级标记语言，创始人是约翰·格鲁伯(John Gruber)。它允许人们 “使用易读易写的纯文本格式编写文档，然后转换成有效的 HTML 文档。
  • Markdown编写的文档后缀为.md, .markdown。

编辑器

  • 本地编辑器: Typora
  • 在线编辑器: 菜鸟工具 Md2All MdEditor
