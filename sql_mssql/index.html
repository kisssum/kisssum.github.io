Mssql数据库笔记

数据库操作

创建数据库

1

create database <数据库名,...>;

移除数据库

1

drop database <数据库名>

进入指定数据库

1

use <数据库名>;

数据表操作

创建数据表

1

create table <数据表名> (字段1 数据类型 约束,...);

创建一个表至少要包含一个字段

移除表

1

drop table <表名>

1

变量

系统全局变量

  • 系统全局变量以@@为前缀，用户不能自定义和改值
变量名 作用
@@SERVERNAME 返回运行 SQL Server 的本地服务器的名称
@@CURSOR_ROWS 返回连接打开的上一个游标中的当前限定行的数目
@@ROWCOUNT 返回受上一语句影响的行数
@@VERSION 返回当前的 SQL Server 安装的版本、处理器体系结构、生成日期和操作系统
@@ERROR 返回执行的上一个 T-SQL 语句的错误号

索引

创建索引

1
2

create [ unique ] [ clustered ] index <索引名>
on <表名>(列名)[asc|desc][,...]
  • unique:唯一的
  • clustered:聚集的

删除索引

1

drop index <索引名> on <表名>

游标

声明游标

1

declare <游标名> cursor for <select 语句>

打开游标

1

open <游标名>

移动游标

1

fetch next from <游标名>

读取游标所在行数据

1

fetch <游标名> into <@接收数据的变量,...>
  • 必须接收全部数据

修改游标数据

1

update <表或视图> set 列名=表达式,... where current of <游标名>

关闭游标

1

close <游标名>

释放游标

1

deallocate <游标名>

备份与还原

备份

创建备份设备

1

sp_addumpdevice '备份设备类型','备份文件名','备份路径'

完整数据库备份

1

backup database <数据库名> to <备份文件名> with name='描述'

差异数据库备份

1

backup database <数据库名> to <备份文件名> with differential,name='描述'

事务日志备份

1

backup log <数据库名> to <备份文件名> with name='描述'

还原

完整数据库还原

1

restore database <还原的数据库名> from <备份文件名> with file=1, norecovery

差异数据库还原

1

restore database <还原的数据库名> from <备份文件名> with file=2, norecovery

事务日志还原

1

restore database <还原的数据库名> from <备份文件名> with file=3

授权

拒绝

1

收回

1

revoke select|insert|delete|update[,...] on <表> from <用户>
